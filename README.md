# Birthday-Reminder

  An app to track and remember the birthdays of your life mates.  
  
# Architeture Pattren: 

  * MVC - Modern Approach

# Desgin Pattrens used in this project: 
  * Coordinator
  * Factory
  * Delegation
  * Dependecy Injection 
  * Service
  * Singleton
  
# Techinolges used in this project:
  * Concurrency
  * SQLite
  * Contacts framework
  * Image Picker
